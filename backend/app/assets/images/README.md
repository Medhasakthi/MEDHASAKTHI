- **Usage**: Certificate watermarks and logo placement
- **Format**: PNG with transparent background
- **Recommended Size**: 512x512 pixels or higher
- **Description**: Your golden MEDHASAKTHI logo with shield and brain/circuit design

## Current Implementation

The certificate generation system will automatically:
1. Load the logo from this directory
2. Apply it as a watermark with configurable opacity and positioning
3. Use it for corner logo placement on certificates
4. Create a placeholder if the file is not found
